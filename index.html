Meus Projetos

  • GreenPlace

    Desenvolvido em grupo para a disciplina de Programação Web, o GreenPlace é uma plataforma de consciencialização ambiental que permite aos utilizadores monitorizarem a sua pegada de carbono através da análise de hábitos diários, fornecendo simultaneamente às empresas um painel de dados sobre a sustentabilidade dos seus clientes. O sistema foi construído com Python e a framework Flask, utilizando uma arquitetura modular de Blueprints e persistência de dados em ficheiros CSV. A interface é totalmente responsiva e dinâmica, integrando HTML5, CSS3 e JavaScript para garantir validações em tempo real e uma experiência de utilização fluida em diversos dispositivos.

  • REsgate.bot

    O projeto integra automação, inteligência artificial e comunicação via apps de mensagem para permitir que qualquer pessoa denuncie rapidamente casos envolvendo animis. Todas as informações coletadas são armazenadas automaticamente em uma planilha do Google Sheets e enviadas por email ao final da denúncia. O propósito é oferecer uma solução simples e eficiente que apoia o resgate, o bem-estar animal e a gestão de ocorrências. O projeto foi desenvolvido com JSON, JavaScript e n8n.

  • Browser CLI

    O projeto simula o funcionamento de um navegador web focado na gestão de histórico de navegação e hierarquia de URLs. Através de uma interface CLI, o sistema valida endereços, mapeia links internos e gerencia o percurso do usuário, permitindo avançar em diretórios ou retornar páginas. O propósito é aplicar estruturas de dados clássicas: uma Pilha para o histórico (LIFO) e uma Árvore para a organização de domínios e suas subpastas. O sistema também realiza a leitura de arquivos físicos para exibir o conteúdo das páginas encontradas. O projeto foi desenvolvido com foco em encapsulamento e modularização, utilizando Python e manipulação de arquivos .txt.
